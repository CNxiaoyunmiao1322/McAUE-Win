# McAUE Launcher

McAUE Launcher is an independent third-party game launcher project.

## Overview

This project aims to provide a modern launcher experience.

Features planned:

- Microsoft account authentication
- Game version management
- Mod management
- User profile support

## Authentication

The launcher uses Microsoft's official OAuth authentication flow.

No user passwords are collected or stored.

Authentication tokens are handled securely through official APIs.

## Compliance

This project does not bypass authentication,
license verification,
or any security mechanisms.

The project follows Minecraft EULA and usage guidelines.

## Status

Early development stage.
